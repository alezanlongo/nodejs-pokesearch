# nodejs-pokesearch
One two one two, it's just a test

*App nodeJS middleware.*
Obtiene resultados desde pokeapi.co y los devuelve a un cliente REST según el keyword pasado por post

Nota:
La primera ejecución, desde NodeJS, se debe llamar a la función fetchPokemonList. Esta recorrerá la lista disponible de Pokemon y creará una repositorio de datos local con el nombre y id de cada personaje.


*App React*
Carpeta: react-app

Contiene el cliente React que consume App NodeJS
